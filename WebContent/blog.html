Traversing a Tree - Preorder Traversal

There are many problems with respect to Computer Engineering that require the traversal of a tree. One of these is DCR, I plan on discussing DCR in the future, but if you're interested you can view a document about this topic on IEEE.

Preorder Traversal is a method of traversing a Tree Data Structure. The idea of the pre-order traversal is to navigate from a node to its left most node. Each node will attempt to go to its left most node. When the algorithm runs out of left nodes, the algorithm will continue by going right and repeating the process. If the traversal is unable to continue going right, it backs up a node and attempts to go right. This continues until the entire tree has been visited. This blog will continue by showing code in Java, illustrating a interactive tree and demonstrating unit tests that can be performed on the code. This can be represented in java as:

public interface Action
{
	public void performAction(Node node);
}
public interface Node
{
	public Node getLeft();
	public Node getRight();
}

/**
 * This traverses nodes in a preorder fashion.
 * It calls nodes recursively from the left and then 
 * recursively calls the nodes to the right.
**/
public void PreorderTraversal(Node head, Action action)
{
	if(head==null)
		return;
	action.performAction(head);
	Node left=head.getLeft();
	PreorderTraversal(left, action);
	Node right=head.getRight();
	PreorderTraversal(right, action);
}

Below is a display of the tree as it goes through the preorder traversal. Press next to iterate through each element. The active element will be yellow, inactive elements will be blue and elements that have already been processed will be grey. You can delete elements by pressing the X button. The tree will rebalance itself when elements are deleted.

void PT(Node head, Action action)
{
  if(head==null)
    return;
  action.performAction(head);
  Node left=head.getLeft();
  PreorderTraversal(left, action);
  Node right=head.getRight();
  PreorderTraversal(right, action);
}
